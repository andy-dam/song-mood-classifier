About the Project

The project allows the user to input any song that's available on Spotify classify it based on 4 different moods:

  1. Happy
  2. Sad
  3. Angry
  4. Relaxed

The model uses pre-labelled data sourced from Kaggle. It employs the Random Forest Classifier model from Scikit Learn. To query songs, it uses the Spotify Web API to search songs, returning audio features used to predict the mood.

Dataset Credits:

https://www.kaggle.com/datasets/abdullahorzan/moodify-dataset
